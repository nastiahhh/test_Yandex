Болевые точки · 3 критичных + 6 значимых
Критичнонаблюдение
Система молча не отвечает — нет ни результата, ни ошибки
При сложном промпте система не выдаёт результат и не сообщает об ошибке. Пользователь видит только иконки лайк/дизлайк под пустым местом. Нет понимания: баг, лимит, таймаут?
Гипотетическое влияние: пользователь, получивший немотивированный пустой ответ, с высокой вероятностью не возвращается в следующей сессии
Критичнонаблюдение
Название шаблона не отображается — только длинный промпт
В поле ввода подставляется полный текст промпта шаблона (5+ строк), но не его название. Пользователь не может быстро подтвердить, тот ли шаблон выбран. При первом опыте это создаёт неуверенность.
Гипотетическое влияние: снижает доверие к интерфейсу на этапе подтверждения выбора — ключевой момент перед первой генерацией
Критичнонаблюдение
Нет Inspiration Hub для генерации фото — только чат
Для видео есть полноценный каталог с шаблонами, Filter of the Day и структурой. Для генерации изображений — только чат и текстовая команда. Асимметрия опыта для двух основных функций.
Гипотетическое влияние: низкий activation rate фичи генерации фото — пользователи не знают о ней или не понимают, как начать
Высокийнаблюдение
Нет иерархии действий под результатом
Кнопки скачивания, шеринга, лайков — одного визуального веса. Главное действие (сохранить/поделиться) не выделено. «Change the style» выглядит как дополнительная опция, не как CTA.
Гипотетическое влияние: снижает % сохранений и шеринга — ключевого канала виральности
Высокийнаблюдение
Смена стиля требует повторной загрузки фото
Итерация по стилям — ожидаемый и частый сценарий. Каждый раз загружать фото заново — лишний шаг, который прерывает творческий flow. Конкуренты этого не требуют.
Гипотетическое влияние: снижение среднего числа итераций на сессию → меньше вовлечения со стилями и premium-контентом
Высокийнаблюдение
Потеря навигации после «Назад» + раздел «Websites»
Кнопка «Назад» не ведёт в предсказуемое место. Раздел «Websites» в навигации непонятен — его назначение нигде не объяснено. Создаёт барьер для возвращающихся пользователей.
Гипотетическое влияние: снижает retention — пользователи не могут найти прошлые результаты и теряют интерес
Среднийнаблюдение
Нет progress bar — только текстовый таймер
«58 seconds left» — конкретно и полезно. Но прогресс-бар (полоса заполнения) создаёт более сильное ощущение движения к результату. Без него ожидание воспринимается тяжелее, особенно при сбоях.
Гипотетическое влияние: рост тревожности при ожидании, особенно при первом использовании
Среднийнаблюдение
Нет fullscreen-просмотра с первого касания
Результат виден в превью внутри чата. Для оценки качества пользователь должен выполнить дополнительное действие, которое неочевидно с первого взгляда.
Гипотетическое влияние: воспринимаемое качество результата ниже реального — негативно влияет на оценку продукта
Среднийнаблюдение
Шаблоны недоступны в существующем чате
Нет способа открыть Inspiration Hub из активного чата. Для использования другого шаблона нужно создавать новый чат, теряя контекст.
Гипотетическое влияние: фрагментация истории работы, лишние чаты, ощущение неудобства у повторных пользователей
A/B-гипотезы · 3 приоритетных эксперимента
H-01Чип с названием шаблона в поле ввода чата
Контроль (сейчас)
В поле ввода — только полный текст промпта шаблона (5+ строк). Название (напр. «Jedi Saber») нигде не показано.
Вариант
Над полем ввода или в нём — небольшой чип «Jedi Saber». Пользователь сразу видит название выбранного шаблона и может его убрать.
Метрика успеха
Рост % пользователей, отправивших запрос после выбора шаблона (без удаления промпта и выхода обратно в каталог)
H-02Inspiration Hub для генерации фото на главном экране
Контроль (сейчас)
Генерация фото доступна только через чат. На главном экране нет карточки или входа.
Вариант
Рядом с «Make Photo Alive» — карточка «Generate Image» с входом в Inspiration Hub для фото (те же категории Popular / Trending, Filter of the Day).
Метрика успеха
Рост activation rate фичи генерации фото (% новых пользователей, выполнивших первую генерацию фото в первые 3 дня)
H-03Смена стиля с сохранённым фото + крупная кнопка скачивания
Контроль (сейчас)
«Change the style» — маленькая кнопка. Требует повторной загрузки фото. Кнопка скачивания — иконка того же веса, что лайк/дизлайк.
Вариант
«Change the style» открывает inline-шторку с шаблонами, фото сохраняется. Кнопка «Сохранить» большая, выделенная, основной CTA под результатом.
Метрика успеха
Рост среднего числа генераций на сессию + рост % пользователей, сохранивших результат

Выводы

Yandex AI имеет реальное конкурентное преимущество в видеогенерации: Inspiration Hub с шаблонами, Filter of the Day, таймер, блок занятости во время ожидания — это продуманные решения, которых нет у многих конкурентов. Когда шаблон применяется корректно, результат качественный.

Проблема не в модели — три критичных разрыва в UX-склейке уничтожают опыт: система не отвечает и не объясняет почему; название шаблона не подтверждается в чате; генерация фото скрыта за чатом без шаблонов. Это не сложные архитектурные задачи — это пропущенные состояния интерфейса.

Первые два исправления (чип шаблона + сообщение об ошибке) — один спринт, нулевые изменения в модели. Они закрывают самые частые точки разочарования до того, как пользователь успевает оценить качество продукта.

Важная оговорка: данное исследование основано на одной сессии тестирования одним пользователем. Для количественного подтверждения гипотез необходимы модерируемые тесты (5–8 участников), анализ funnel drop-off и описанные A/B-эксперименты.
